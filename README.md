# تطبيق متجر الأزياء باستخدام Flutter

## الوصف

هذا التطبيق هو مشروع مبني باستخدام Flutter يوضح أساسيات التنقل بين الشاشات (Navigation) وإدارة البيانات. يتيح للمستخدم تصفح مجموعة من عناصر الأزياء، عرض تفاصيل كل منتج، وإضافته إلى قائمة العناصر المحفوظة بسهولة.

## المميزات

* التنقل بين الشاشات باستخدام Navigator
* إرسال واستقبال البيانات بين الصفحات
* عرض قائمة منتجات (ملابس) مع صور وأسعار
* صفحة تفاصيل لكل منتج
* إمكانية حفظ العناصر في قائمة خاصة
* واجهة مستخدم بسيطة وجذابة بألوان متناسقة
* واجهة مستخدم بسيطة وجذابة بألوان متناسقة
